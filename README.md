# 문항 변천사 뷰어

여러 버전의 시험지(`.hwp` / `.hwpx`, 선택적으로 `.pdf`)를 넣으면 **문항을 자동으로 매칭**해 버전별 변화를 한눈에 보여주는 단일 HTML 도구입니다. 모든 처리는 **브라우저 안에서만** 이루어지며 파일은 어디에도 전송되지 않습니다.

## 바로 사용하기

👉 **https://hong-jame.github.io/munhang-history-viewer/**

파일을 설치할 필요 없이 위 링크에서 바로 쓸 수 있습니다. (그림·표가 정확히 그려지려면 인터넷 연결이 필요합니다 — 웹폰트와 PDF 처리 라이브러리를 불러옵니다.)

## 주요 기능

- **문항별 변천사**: 초기본 → 최종본까지 버전별 변화를 단어 단위 diff로 표시
- **문항 변천 타임라인**: 어느 버전에서 무엇이(지문·배점·그림·번호) 바뀌었는지 가로 타임라인으로
- **변경 요약 한 줄**: 문항마다 "배점 3.4→4.0점 · 그림 1건 변경 · 문구 수정 2회"처럼 자동 요약
- **배점·보기·선택지 변경 감지**, **한글 내장 그림 변경 감지**(이전/새 그림 비교 + 차이 겹쳐 보기)
- **문항 번호 변경 추적** + 최종본 기준 정렬
- **전체 변경 맵**(문항×버전 매트릭스), **검토 기록**, **검토 보고서(인쇄/PDF)**, **변경 이력 CSV 내보내기**
- 병합 셀·`<보기>` 상자·표를 원본에 가깝게 복원

## 사용법

1. 왼쪽에 버전 파일들을 끌어다 놓기 (여러 개 동시 선택 가능)
2. 목록에서 순서를 초기본 → 최종본으로 정렬 (↑↓)
3. **분석하기** 클릭

## 지원 형식

- `.hwp` (OLE 복합문서), `.hwpx` (ZIP) — 텍스트·표·내장 그림 복원
- `.pdf` — 텍스트 PDF는 문항 단위 비교, 스캔(이미지) PDF는 페이지 단위 비교
